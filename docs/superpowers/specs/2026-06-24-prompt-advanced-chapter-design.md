# 第 21 章 Prompt Engineering 进阶设计说明

## 目标

为当前 Python、FastAPI、RAG 学习项目增加第 21 章，使学习者能够把提示词从“自然语言说明”升级为可验证的 AI 接口契约，并亲手完成一次独立实现。

本章结束时，学习者应能回答并实践：

1. Prompt 能约束模型到什么程度，不能保证什么。
2. Pydantic Schema、模型结构化输出和普通 JSON 提示词有什么区别。
3. Few-Shot 示例如何影响模型输出模式。
4. `temperature` 与 `top_p` 如何影响采样，为什么通常只重点调整一个。
5. Prompt Injection 为什么不能只靠 System Prompt 防御。
6. 如何在当前 FastAPI 项目中实现并验证一个结构化输出接口。

## 产物

1. `md/21_Prompt_Engineering进阶.md`
   - 使用现有教程的关卡式结构。
   - 包含一句话模型、准确术语、边界、最小模板、常见坑和检查点。
   - 包含“读懂、跟写、独立重写”三遍学习流程。
2. `routers/prompt_advanced_router.py`
   - 使用项目现有的 `ChatDeepSeek` 与环境变量配置方式。
   - 提供一个最小结构化输出接口。
   - 将输入模型、输出模型、Prompt 和路由职责分开。
3. `test/test_prompt_advanced.py`
   - 不调用真实大模型。
   - 使用替身或依赖边界验证请求、响应 Schema 和错误处理。
4. `main.py`
   - 只增加新路由注册，不重构现有路由。

## 教学结构

### 第一关：Prompt 不等于程序约束

通过“要求模型返回 JSON”与“程序验证 Schema”对比，建立核心边界：Prompt 是概率性指令，Schema 校验是确定性程序检查。

### 第二关：Pydantic 结构化输出

展示请求经过 Prompt、模型、结构化解析和 Pydantic 校验后成为 FastAPI JSON 响应的数据流。给出可以直接复制的最小实现。

### 第三关：Few-Shot 示例

使用一个零样本输入和两个少样本示例，说明示例是在演示任务模式，不是把训练数据写入模型。

### 第四关：采样参数

准确区分 `temperature`、`top_p` 和确定性业务输出。章节不要求背数学公式，只要求能根据“抽取、分类、创作”选择合理方向。

### 第五关：Prompt Injection 边界

区分系统指令、用户输入和外部检索内容。明确输入分隔、权限控制、工具白名单、输出校验和人工确认的职责，不宣传“万能防注入提示词”。

### 第六关：项目可运行路由

逐步实现一个文本任务提取接口，输入自然语言，输出固定字段，例如标题、优先级和标签。接口通过 Pydantic 返回可验证 JSON。

### 第七关：独立重写

换成“用户反馈分类”场景，只提供：

- 接口需求；
- 输入和输出字段；
- 完成步骤；
- curl 或 pytest 验证标准。

不在该环节提前提供完整实现。学习者完成后再进行代码检查和针对性补漏。

### 第八关：检查点与调试

用四条理解标准验收，并覆盖模型不支持结构化输出、返回字段不合法、环境变量缺失和测试误调用真实模型等常见问题。

## 代码边界

- 新路由独立存在，不修改现有 `routers/ai_router.py` 和 `routers/langchain_rag_router.py` 的行为。
- 模型凭证继续来自 `.env`，不得写入源码。
- Pydantic 负责应用层 Schema；模型提供商的原生结构化输出能力负责尽量生成匹配内容，两者不能混为一层。
- 测试必须隔离外部模型调用，避免费用、网络波动和随机性。
- Prompt Injection 内容只覆盖本章所需防线；完整安全治理留到后续 `ai-safety` 阶段。

## 错误处理

- 输入不满足请求 Schema：由 FastAPI 返回 422。
- 模型或结构化解析失败：路由转换为稳定的上游服务错误，不把密钥、完整 Prompt 或内部堆栈返回给客户端。
- 环境变量缺失：在可定位的位置给出明确配置错误。

## 验收标准

1. 第 21 章符合项目教学规范并包含八个关卡。
2. 新路由能被 FastAPI 注册并出现在 `/docs`。
3. 示例接口的响应满足声明的 Pydantic Schema。
4. 自动测试不访问真实模型且能够验证成功与失败路径。
5. 独立实战没有泄露完整答案，并提供可以马上执行的验证方法。
6. 原有测试没有因路由注册而回归。

# Python 语言心智模型桥接课程设计

## 目标

基于用户提供的通用编程语言文档的主题范围，原创编写适合 Python 初学者的四篇桥接课。课程放在 Python 基础和 Git 课程之间，帮助学习者理解 Python 程序如何运行、表达式与类型、函数进阶以及类与对象。

## 来源与改写边界

来源为 <https://github.com/write-magic-code/public/blob/main/language/language.md>。只借鉴其主题覆盖范围，不复制其文字、跨语言示例、插图或叙事风格。所有代码示例、解释、练习和章节组织均为 Python 初学者重写。

## 文件与顺序

新增：

```text
15_Python程序如何运行_源码解释器与报错.md
16_Python表达式_变量绑定与真值判断.md
17_Python函数进阶_Lambda与闭包.md
18_Python类_对象与实例.md
```

现有 Git 至 AI 地图的课程整体顺延四位：原 `15–23` 变为 `19–27`。README、前后章节的下一步链接和课程总顺序同时更新。

## 章节边界

| 章节 | 教学内容 | 明确不教 |
| --- | --- | --- |
| 15 | 源码、语法、Python 解释器、进程、语法错误与运行时异常 | CPython 源码、字节码优化、编译器实现 |
| 16 | 表达式、变量绑定、返回值、`None`、真值判断、显式类型转换 | C/C++ 左右值、指针与内存地址管理 |
| 17 | 函数是一等对象、简单 lambda、闭包捕获外层名字 | 函数式编程范式、复杂装饰器实现 |
| 18 | 类、实例、属性、方法、`__init__`、资源释放的 Python 直觉 | 手动内存管理、析构器设计、继承体系 |

同步/异步只在第 15 章作为后续主题提示，详细学习仍放在项目既有异步课程。

## 教学合同与验收

每篇新增课包含一句话理解、范围、术语、最小 Python 模板、常见坑、检查点、小练习与下一步。全部 Python 代码块必须通过语法编译；链接不得指向旧编号；新增内容不得包含模型训练或跨语言必需知识。

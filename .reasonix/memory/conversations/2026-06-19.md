### 最终掌握结论

```text
migration = 一次结构变化脚本。
migrations = 一系列结构变化历史。
backup = 当前数据库完整状态备份，包含真实数据。
```

```text
stamp head = Alembic 命令；不执行 SQL，只把已有数据库标记为最新 revision。
alembic_version = 数据库版本记录表；记录当前数据库处在哪个 revision。
revision = migration 的版本号；alembic revision 是创建 migration 文件的命令。
```

## 对话 25: BaseModel 与 SQLAlchemy Base 辨析

**时间**: 2026-05-31
**用户任务**: 理解 BaseModel 和 Base 的区别，为什么数据库相关类要用它们

### 核心对比

| 类 | 来源 | 作用 | 比喻 |
|:---|:---|:---|:---|
| `BaseModel` | **Pydantic** | 校验数据格式 + 自动转 JSON | 快递单模板 |
| `Base` | **SQLAlchemy** | 把 Python 类映射成数据库表 | 藏书编号系统 |

### 为什么用 BaseModel？

不用 BaseModel 要手写 20 行类型检查，用了自动校验 + 自动转 JSON。

### 为什么用 SQLAlchemy Base？

不用要手写 SQL（INSERT/SELECT/UPDATE），用了操作 Python 对象就行。

### 常见错误

- `class Document(BaseModel)` ❌ → 应该用 `Base`
- `class SearchResult(Base)` ❌ → 应该用 `BaseModel`

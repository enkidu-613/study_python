### 考前薄弱点

- `create_agent(...)` 和 `agent.invoke(...)` 的阶段区别。
- `tools=[@tool 函数]` 与第 26 章 `TOOLS + TOOL_FUNCTIONS` 的边界。
- Agent Tool 输出最好是短字符串。
- `AIMessage` 最终答案用 `.content`。
- 有 `checkpointer` 时，每次 invoke 都要传同一个 `thread_id` config。
- `state["messages"]` 是 Agent 的消息轨迹，比传统聊天历史更宽。

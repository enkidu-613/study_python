# 02. 安装编辑器并创建第一个项目

> **一句话理解：编辑器是你写、查看和调试项目文件的工作台。**

## 学什么，不学什么

学：安装一个编辑器、打开文件夹而不是单个文件。  
不学：Python 安装；下一章处理。

## 术语

| 术语 | 含义 |
| --- | --- |
| 项目 | 为同一目标放在一个文件夹中的代码和配置。 |
| 文件树 | 编辑器左侧显示项目文件夹层次的区域。 |
| 集成终端 | 编辑器中可输入命令的终端窗口。 |

## 最小模板

到 PyCharm 或 VS Code 官方网站下载与你系统匹配的版本并安装。打开后选择 **Open Folder / 打开文件夹**，选择上一章的 `ai-beginner`，新建 `hello.py`。

```python
print("你好，编程！")
```

保存后，文件树应显示 `ai-beginner` 和其下的 `hello.py`。

## 常见坑

- 双击单个文件而未打开项目文件夹。
- 安装多个编辑器后不知道当前在哪个写代码；只保留一个作为本课程工作台。
- 未保存就运行；先确认标签页没有未保存标记。

## 检查点

- [ ] 编辑器文件树能看到 `hello.py`。
- [ ] `hello.py` 中有一行 `print(...)` 并已保存。

# AI 应用开发：零基础自学前置课程

这是一条给完全新手的文字自学路线：不需要截图、不需要视频，也不需要先懂编程。每章只完成一个动作；完成检查点再读下一章。

| 设备 | 终端 |
| --- | --- |
| Windows 10/11 | PowerShell，不要求 WSL |
| macOS | Terminal（zsh） |
| Linux | Terminal（Bash） |

遇到系统命令时，只复制自己系统对应的一列；Python 代码、Git 概念与练习三端相同。

## 顺序

`00–06` 建立并运行程序 → `07–18` 写、理解与调试 Python → `19–23` 保存并上传代码 → `24–26` 理解 Web 与数据 → `27` 进入现有 AI 应用主线。

完成后从 [项目环境配置](../00_环境配置与PyCharm使用.md) 开始。不要跳过 `03`、`06`、`13`、`15–23`：它们解决新手最常见的“为什么跑不起来/为什么传不上去”。

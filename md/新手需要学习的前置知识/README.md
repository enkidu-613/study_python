# AI 应用开发新手前置知识

这是一条给“会用电脑，但从未学过编程”的独立起步路线。目标不是训练模型，而是尽快具备开发、调试并交付 AI 应用的基础。

## 学习顺序

按文件名从 `00` 读到 `11`。不要跳过 `02`、`06`、`07`、`08`：它们决定你遇到报错、协作和部署时能否自己定位问题。

| 阶段 | 文件 | 预计专注学习时间 | 产出 |
| --- | --- | ---: | --- |
| 导航 | 00 | 30 分钟 | 知道 AI 应用工程师做什么 |
| 电脑与终端 | 01–02 | 4–6 小时 | 能创建、定位、运行文件 |
| 编程与 Python | 03–06 | 20–30 小时 | 能完成一个小 Python 项目 |
| 工程与网络 | 07–10 | 12–18 小时 | 能管理代码、理解 API 与数据 |
| AI 应用地图 | 11 | 1 小时 | 知道下一步为什么学 RAG、Agent |

时间只作参考。每章达到检查点再继续，不靠“看完了”算完成。

## 每章怎么学

1. 先读“一句话理解”和“本章不学什么”。
2. 把最小模板亲手运行一次。
3. 改一处内容并观察结果。
4. 完成检查点和小练习。

## 完成后的入口

完成 `11` 后，进入现有项目的 [环境配置与 PyCharm 使用](../00_环境配置与PyCharm使用.md)，再按项目 `md/README.md` 的主线学习 Python、FastAPI、RAG、Agent、评估与部署。

## 这条路线不包含什么

它不教模型从零训练、数学推导或算法岗位刷题。你将学习的是：如何可靠地使用预训练模型与 API，做出能被用户使用的 AI 产品。

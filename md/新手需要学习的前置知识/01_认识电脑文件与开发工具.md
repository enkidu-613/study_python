# 01. 认识电脑文件与开发工具

> **一句话理解：代码是文本文件；编辑器负责写它，Python 解释器负责运行它。**

## 学什么，不学什么

学：文件、文件夹、扩展名、编辑器和解释器。  
不学：安装步骤；下一章处理。

## 术语

| 术语 | 含义 |
| --- | --- |
| 文件 | 保存内容的单位，如 `hello.py`。 |
| 文件夹/目录 | 装文件的容器。 |
| 编辑器 | 写代码的程序，如 PyCharm、VS Code。 |
| 解释器 | 读取并执行 `.py` 文件的 Python 程序。 |

## 最小模板

```text
ai-beginner/
└── hello.py
```

`hello.py` 是普通 UTF-8 文本，不是 Word 文档。`.py` 告诉人和工具“这是 Python 源代码”。

## 常见坑

- 文件实际叫 `hello.py.txt`。
- 把编辑器当作 Python；两者必须分别安装。
- 把下载文件夹当项目目录；项目应有自己的文件夹。

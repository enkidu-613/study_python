# 04. 路径、文件夹与文件扩展名

> **一句话理解：路径是电脑找到文件的地址；相对路径以当前目录为起点。**

## 学什么，不学什么

学：绝对路径、相对路径、扩展名。  
不学：终端命令；下一章使用路径。

## 术语

| 术语 | 含义 |
| --- | --- |
| 绝对路径 | 从系统位置开始的完整地址。 |
| 相对路径 | 从当前目录开始的地址。 |
| 当前目录 | 命令现在默认操作的文件夹。 |

## 最小模板

| 系统 | `hello.py` 的绝对路径示例 |
| --- | --- |
| Windows | `C:\Users\你的用户名\Documents\ai-beginner\hello.py` |
| macOS | `/Users/你的用户名/Documents/ai-beginner/hello.py` |
| Linux | `/home/你的用户名/ai-beginner/hello.py` |

在 `ai-beginner` 里写 `hello.py` 是相对位置；从电脑根位置描述它是绝对位置。

## 常见坑

- Windows 用 `\`，macOS/Linux 用 `/`。
- 路径含空格时命令中通常需加引号；新手项目名先用英文小写和连字符。
- 移动文件后旧路径会失效。

## 检查点

- [ ] 能找到 `hello.py` 的完整位置。
- [ ] 能解释相对路径为什么需要当前目录。

## 小练习

新建 `data` 文件夹和 `data/note.txt`，用文件树确认层级。

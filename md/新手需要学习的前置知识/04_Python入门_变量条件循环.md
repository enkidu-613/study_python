# 04. Python 入门：变量、条件与循环

> **一句话理解：Python 程序用变量保存数据，用条件选路线，用循环重复处理同类事情。**

## 本章学什么，不学什么

学：变量、字符串/数字/布尔值、`if`、`for` 与 `input`。

不学：函数、列表和字典；它们留到下一章。

## 准确术语

| 术语 | 准确含义 |
| --- | --- |
| 变量 | 有名字的值，例如 `name = "小明"`。 |
| 字符串（`str`） | 文本值，通常用引号包住。 |
| 整数（`int`） | 不带小数的数字。 |
| 布尔值（`bool`） | 只有 `True` 或 `False` 的真假值。 |
| 条件表达式 | 计算后得到真假的表达式，例如 `age >= 18`。 |
| 循环 | 对一组项目重复执行同一段代码。 |

## 最小模板：读取一个名字并问候

新建 `greet.py`：

```python
name = input("请输入你的名字：")

if name == "":
    print("你还没有输入名字")
else:
    print(f"你好，{name}！")
```

`input()` 是 Python 内置函数：它显示提示文字，等待用户输入，并且**总是返回字符串**。`f"...{name}..."` 是 f-string，会把变量值放进文本。

## 最小模板：循环处理固定任务

```python
tasks = ["学习 Python", "喝水", "运行代码"]

for task in tasks:
    print(f"现在做：{task}")
```

`tasks` 是一个列表；`for` 每轮取出一个项目放进 `task`，然后执行缩进的代码块。

## 常见坑

- 缩进不一致：Python 用缩进表示代码块，通常统一为四个空格。
- 把 `=` 和 `==` 混淆：`=` 赋值，`==` 比较是否相等。
- 以为 `input()` 返回数字：需要时写 `int(input("请输入年龄："))`，但非法输入会报错。
- 字符串忘记加引号：`"你好"` 是文本，`你好` 会被当成名字查找。

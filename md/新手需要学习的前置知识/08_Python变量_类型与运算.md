# 08. Python 变量、类型与运算

> **一句话理解：变量给数据取名；类型决定数据能做什么运算。**

## 学什么，不学什么

学：`str`、`int`、`float`、`bool` 与基础运算。  
不学：条件和循环。

## 术语

| 类型 | 示例 |
| --- | --- |
| `str` | `"课程"` |
| `int` | `3` |
| `float` | `3.5` |
| `bool` | `True` |

## 最小模板

```python
name = "小明"
hours = 2
total = hours * 60
finished = total >= 60
print(name, total, finished)
```

右边先计算，再把值保存到左边变量。`=` 是赋值，`==` 才是比较相等。

## 常见坑

- `"2" + 1` 不能计算；字符串与数字需先转换。
- 把 `=` 写进判断。
- 变量名以数字开头。

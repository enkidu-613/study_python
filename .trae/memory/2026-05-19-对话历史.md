## 对话 11: 余弦相似度深度讲解

**时间**: 2026-05-19
**用户问题**: 为什么用 `np.dot` 和 `np.linalg.norm` 计算余弦相似度？有什么公式？

**深入讲解**:
1. **公式拆解**
   - 点积：`np.dot(a, b)` = |A| × |B| × cos(θ)
   - 范数：`np.linalg.norm(a)` = √(Σa²) = 箭的长度
   - 除法：消掉长度，只剩 cos(θ)

2. **电影评分类比**
   - 场景：你和朋友给漫威/文艺片/恐怖片打分
   - 问题：不能比绝对分数（5分制 vs 10分制）
   - 解决：比"口味比例"，不比"绝对数值"

3. **手电筒比喻**
   - 两个手电筒照墙
   - 余弦相似度 = 光斑重叠程度
   - `norm` = 把光斑调成一样大，公平比较

**关键知识点**:
- 归一化：把向量截成单位长度（只看方向，不看长短）
- 余弦范围：1（完全相同）→ 0（垂直无关）→ -1（相反）
- 实际应用：Embedding 都是正数，结果在 0~1 之间

**用户反馈**:
- 数学基础较弱，需要更详细的逐步解释
- 对"归一化"和"消掉长度"概念需要更多直观理解

为 `get_embedding()` 和 `cosine_similarity()` 添加了数学原理注释：

```python
def cosine_similarity(a, b):
    """
    数学公式：
         A · B        Σ(aᵢ × bᵢ)
    cos(θ) = ───── = ────────────────────────
           ||A|| × ||B||   √(Σaᵢ²) × √(Σbᵢ²)
    """
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))
```

### 5. ModelScope API 接入

最终成功配置：

```python
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

MODELSCOPE_API_KEY = os.getenv("MODELSCOPE_API_KEY")

client = OpenAI(
    base_url="https://api-inference.modelscope.cn/v1",
    api_key=MODELSCOPE_API_KEY
)

def get_embedding(text: str) -> list[float]:
    response = client.embeddings.create(
        model="Qwen/Qwen3-Embedding-8B",
        input=text,
        encoding_format="float"
    )
    return response.data[0].embedding
```

### 6. 为什么 Embedding 需要网络请求？

解释了 Embedding 的本质：
- Embedding 是**训练好的神经网络**（不是简单计算）
- 需要理解语义：比如"代码质量"和"可维护性"是相关的
- 在线 API：效果好、免费额度够用、响应快
- 本地模型：离线可用，但需要下载大模型

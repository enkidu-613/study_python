### 考前薄弱点

- `create_agent(...)` 和 `agent.invoke(...)` 的阶段区别。
- `tools=[@tool 函数]` 与第 26 章 `TOOLS + TOOL_FUNCTIONS` 的边界。
- `@tool` docstring 会变成模型可见工具描述，不是普通 `#` 注释。
- Agent Tool 输出最好是短字符串，不要直接把 `list[Document]` 扔给模型。
- `result["messages"][-1]` 是 `AIMessage` 对象，最终答案用 `.content`。
- 有 `checkpointer` 时，每次 invoke 都要传同一个 `thread_id` config 才能恢复同一对话。
- `state["messages"]` 是 Agent 的消息轨迹，可能包含聊天和工具步骤，比传统聊天历史更宽。

## 对话 30: 上下文窗口管理学习与测试

**时间**: 2026-06-03
**用户任务**: 进行学习计划下一步（上下文窗口管理阶段）

**主要内容**:
- 明确上下文窗口是 LLM 的短期记忆容量，必须管理
- 三层防御策略：
  1. `MAX_CONTEXT_TOKENS=2000`：资料片段上限（`build_system_prompt` 中截断）
  2. `MAX_INPUT_TOKENS=6000`：总输入上限（System + User 检查）
  3. `MAX_OUTPUT_TOKENS=2000`：回答长度上限（API 参数）
- Token 估算使用 `tiktoken`（cl100k_base），不能用 Embedding 模型计数
- 截断策略用 `break` 而非 `continue`（results 已按相似度排序）
- 防幻觉：Prompt 工程给 AI "拒绝权" + 相似度阈值过滤

**关键代码**:
```python
MAX_CONTEXT_TOKENS = 2000
MAX_INPUT_TOKENS = 6000
MAX_OUTPUT_TOKENS = 2000

def estimate_tokens(text: str, model: str = "cl100k_base") -> int:
    encoding = tiktoken.get_encoding(model)
    return len(encoding.encode(text))

# 资料片段截断
def build_system_prompt(results, max_context=MAX_CONTEXT_TOKENS):
    for r in results:

        if current_tokens + block_tokens > max_context:
```

## 对话 31: 无状态 RAG 特性讨论

**时间**: 2026-06-03

**核心结论**:
- 当前 RAG 是"金鱼记忆"，每次 POST `/chat` 都是全新请求
